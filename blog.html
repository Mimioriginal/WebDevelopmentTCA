Starting with HTML and CSS

11/APR/2022

At first it seemed like a website with 5 pages will be a lot of work. However as always, it's best to start from the beginning and not do everything at once.

It is best to start by creating a new text file, with .html extention (the index.html file for the homepage) and make sure that it has the correct starting tags at the start of the document(DOCTYPE, html tag, link tags (for stylesheets or shortcut icon), title etc).

Once I had the homepage 'head' set up, I could start adding content to my website.

Adding content to the HTML files

12/APR/2022

Once I had the necessary HTML tags ready, it was time to think about what content I wanted to add. The trick was to use the correct html tags for the correct elements.

A great source of information for learning the basics were the Mozilla Developer Guides.

Another thing that helped me make sure that my html and css files were valid was the W3 Markup Validator Service.

I didn't start committing changes to github until I was sure what I wanted my website to look like, so I worked on my back up files first. This way the history wouldn't look messy.

Once I had the contents ready in my html files, it was time to work on the layout, colours and styles of my pages.

Styling with CSS

13/APR/2022

Firstly I created the CSS files by clicking on the css link in the head section of my HTML file. What I love about Visual Studio Code is that it lets me CTRL+click on the stylesheet link and it will automatically create the required CSS file. That way I didn't need to worry about absolute URLs, as it automatically creates the CSS file in the correct folder.

The grid layout is quite interesting because of how much is achievable with it. It took me longer than I'd like to admit to figure out how to divide the elements (such as body, nav, main etc) into grid areas, however after a ton of trials & errors and many long winded youtube videos later I was finally able to achieve and realize what I had envisioned.

My favorite topic was the CSS keyframe animation. It seemed complicated at first, however its quite fun to be able to animate different elements. During my time exploring different ideas I found some amazing animations by some very creative and talented people. As a gamer, my favorite one of course is the Gradient Border.
